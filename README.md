This project performs Exploratory Data Analysis (EDA) on the global layoffs dataset using MySQL. The goal is to analyze trends in layoffs across companies, industries, countries, and time periods.

Project Objectives

- Import and clean the layoffs.csv dataset

- Explore layoff patterns using SQL

Identify:

- Companies with the highest layoffs

- Industries most affected

- Layoff trends by year and month

- Country-wise distribution

- Analyze percentage layoffs vs total layoffs

Files in This Repository:

- eda_queries.sql     → All SQL queries used for analysis
- layoffs.csv         → Raw dataset used in the project
- README.md           → Project documentation
